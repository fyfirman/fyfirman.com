## Array & Map

### For Loop

```go
for i := 0; i < 10; i++ {
    fmt.Println(i)
}
```

### Array Length

```go
arr := []int{1, 2, 3}
fmt.Println(len(arr))
```

### Array Push (Append)

```go
arr := []int{1, 2, 3}
arr = append(arr, 4)
```

### Assign Value to Map

```go
m := make(map[string]int)
m["key"] = 1
fmt.Println(m)
```

### Print Struct

```go
log.Printf("%+v\n", res)
fmt.Printf("%+v\n", res)
```
