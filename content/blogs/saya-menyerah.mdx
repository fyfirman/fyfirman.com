30 September 2022, saya menulis artikel ini sekaligus saya mendeklarasikan untuk menyerah. **Saya menyerah untuk menulis blog**.

Satu bulan yang lalu saya bercita-cita untuk mempunyai sebuah blog untuk mencurahkan hati dan pikiran yang punya. Saya sangat senang untuk membuat suatu hal menjadi keren, termasuk website dan blog ini. Saya bercita-cita untuk menyelesaikan blog dengan fitur blog list dan detail dan mempunyai minimal tiga artikel di dalamnya, tetapi semua itu tidak mudah.

Segala hal tersebut butuh konsistensi. Dalam hal ini saya gagal untuk bertahan & bersabar untuk mengerjakan project ini. Sudah satu bulan rencana ini saya inginkan tetapi untuk membuat dua artikel lainnya adalah sulit. Terlebih lagi saya ingin mempunyai tiga artikel berbahasa inggris. Saya terlalu naif untuk mempunyai keinginan tersebut.
