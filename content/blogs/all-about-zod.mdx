## What is Zod?

Zod is a TypeScript-first schema declaration and validation library. I'm using the term "schema" to broadly refer to any data type, from a simple string to a complex nested object.

## Zod Snippets

### One Variable

```ts
const myVar = "";
z.string().parse(myVar);
```

### Is ISO Date string

```ts
z.string().datetime();
```
